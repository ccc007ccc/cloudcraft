# CloudCraft 未来方向规划

本文用于记录 CloudCraft 未来向“云之维度”发展的设计方向。它不是当前版本承诺，也不要求一次实现；后续可以继续补充地形、生态、实体、建筑、Boss、进度、科技线等内容。

当前阶段只规划云种类、云生物群系、不同云的机制，以及围绕云维度可以展开的基础方向。

## 设计中心

CloudCraft 的长期核心不是只添加几种方块，而是围绕“云”构建一个可探索、可生存、可成长的维度。

这个维度应该有自己的层级结构、天气逻辑、移动方式、资源循环和危险来源。云不只是装饰，也不只是普通方块换皮；不同云应对应真实云型，并把现实中的形态、所在高度、天气含义转化为游戏机制。

## 分类参考

本规划参考两类资料：

- Wikipedia “云”条目：按高度、形态、成因、天气意义整理云的基础知识。
- WMO International Cloud Atlas：按云属、云种、变种、补充特征、附属云、母云和特殊云整理云分类。

适合游戏化的核心结构：

1. 云属：作为主要方块/生态/机制分类。
2. 云种：作为同一云属下的形态分支。
3. 变种：作为视觉或稀有词缀。
4. 补充特征：作为天气、危险、状态或事件机制。
5. 附属云：作为伴生结构、实体、掉落物或局部地形。
6. 特殊云：作为特殊生物群系、事件或维度边界内容。

## 云之维度的层级

云维度可以按高度和危险度分层，而不是做成单一平面。

### 高云层

对应现实中的高云族，主要由冰晶感、纤维感、薄幕感组成。

适合体验：轻盈、开阔、稀薄、风大、资源稀有、视野好但落脚点少。

候选云属：

- 卷云 / Cirrus
- 卷积云 / Cirrocumulus
- 卷层云 / Cirrostratus

### 中云层

对应中云族，云块更厚，形态更复杂，是探索与资源的中间层。

适合体验：稳定探索、云岛群、波状地形、过渡资源、天气前兆。

候选云属：

- 高积云 / Altocumulus
- 高层云 / Altostratus

### 低云层

对应低云族，厚重、灰暗、铺展，接近雾和雨。

适合体验：低能见度、缓慢移动、潮湿、持续消耗、适合隐藏生物或遗迹。

候选云属：

- 层云 / Stratus
- 层积云 / Stratocumulus
- 雨层云 / Nimbostratus

### 直展云层

对应垂直发展的云，贯穿多个高度层，是云维度的危险核心。

适合体验：高耸结构、强对流、雷暴、向上探索、Boss 或大型事件。

候选云属：

- 积云 / Cumulus
- 积雨云 / Cumulonimbus

## 云种类发展方向

### 积云 / Cumulus

当前已有基础积云，可作为云维度的起点云。

定位：基础、常见、可塑性强。

机制方向：

- 普通云平台。
- 可被踩碎，掉落云碎片。
- 可被药水或效果在气态和固态之间转化。
- 作为合成其他云的基础材料。

生物群系方向：

- 积云原野
- 漂浮云岛
- 云丘群

### 层云 / Stratus

定位：低层、铺展、雾状、遮蔽。

机制方向：

- 让水平速度自然趋近较低目标速度。
- 降低视野或制造雾感。
- 适合作为隐藏区域、迷雾通道、低层云海。
- 接近地面时可与“雾”机制关联。

生物群系方向：

- 层云雾海
- 低空白幕
- 雾隐云原

### 卷云 / Cirrus

定位：高空、纤细、冰晶、轻快。

机制方向：

- 保留较高水平漂移速度，不强行刹停。
- 可作为高速移动路线或高空风道。
- 适合做轻薄、透明、易碎的云。
- 可产生冰晶类材料。

生物群系方向：

- 卷云高原
- 冰晶云带
- 高空风径

### 高层云 / Altostratus

定位：中层、灰白幕状、遮光、天气前兆。

机制方向：

- 中等减速，介于层云和卷云之间。
- 降低光照，影响植物或云维度能量系统。
- 可作为雨层云或积雨云生成前兆。
- 适合大面积铺展地形。

生物群系方向：

- 高层云幕
- 灰白天穹
- 暗光云原

### 雨层云 / Nimbostratus

定位：厚重、暗灰、持续降水。

机制方向：

- 水平速度趋近更低目标速度。
- 产生持续雨、水滴、潮湿状态。
- 可以让火焰熄灭、影响视野、让部分云变重。
- 可作为水资源来源，但探索节奏更慢。

生物群系方向：

- 雨层云海
- 长雨云幕
- 潮湿低云层

### 积雨云 / Cumulonimbus

定位：强对流、雷暴、垂直发展、危险核心。

机制方向：

- 不是简单“往上弹飞”，而是表现为强扰动和不稳定。
- 可周期性产生雷、电荷、阵雨或短时强风。
- 可以作为维度大型结构、灾害事件或 Boss 区域。
- 与砧状云、乳状云、墙云、尾云等补充特征联动。

生物群系方向：

- 积雨云塔
- 雷暴云墙
- 对流核心

### 卷积云 / Cirrocumulus

定位：高空、细小颗粒、鱼鳞状。

机制方向：

- 由许多小云块组成，适合碎片化平台。
- 可以作为高空跳跃路线或稀有资源带。
- 适合生成群聚小型云生物。

生物群系方向：

- 鱼鳞云带
- 高空碎云群

### 卷层云 / Cirrostratus

定位：高空薄幕，常产生日晕/月晕视觉。

机制方向：

- 极薄、低碰撞或半透明。
- 可影响光照、天空颜色、晕环视觉。
- 适合作为云维度顶部边界或特殊天象区域。

生物群系方向：

- 晕光云幕
- 薄冰天盖

### 高积云 / Altocumulus

定位：中层块状、波状、成群排列。

机制方向：

- 适合做规律排列的云岛、波浪地形。
- 可做“周期性起伏”的移动或速度变化机制。
- 与波状云、荚状云、堡状云词缀联动。

生物群系方向：

- 高积云群岛
- 波状云原
- 荚状云丘

### 层积云 / Stratocumulus

定位：低层大片团块，介于层云和积云之间。

机制方向：

- 比层云更稳定，比积云更铺展。
- 适合做防线、厚云地形、低空云大陆。
- 可作为低层主要建筑地基。

生物群系方向：

- 层积云大陆
- 低空云垒

## 云种、变种与词缀

WMO 的云种和变种可以做成后续词缀系统，而不是一开始就全部变成独立方块。

### 适合做形态词缀的云种

- 毛状云：纤细、风向感强，可提升移动或风属性。
- 钩状云：末端弯曲，可做牵引、钩风或路径改变。
- 密云：厚度更高，可提升承载和防护。
- 堡状云：像塔一样向上突起，适合生成哨塔或敌对结构。
- 絮状云：松散破碎，适合分裂、掉落、召唤小云。
- 荚状云：稀有云丘，可作为特殊资源点。
- 滚卷云：横向移动云墙，可做区域推进事件。
- 薄幕状云：轻薄覆盖，适合光照和隐匿机制。
- 碎云：破碎残片，可做副产物和小型平台。

### 适合做环境词缀的变种

- 透光云：保留光照，适合安全区域。
- 漏隙云：有孔洞，适合穿行、垂直通路。
- 蔽光云：遮光，适合阴暗生态或怪物生成。
- 复云：多层叠加，适合厚云地形。
- 波状云：速度或高度呈周期变化。
- 辐状云：引导路径向某个中心汇聚。
- 网状云：形成网格状阻隔或陷阱。

### 适合做事件特征的补充特征

- 降水性云：雨、雪、冰粒、水资源。
- 幡状云：雨滴未落地就蒸发，可做“虚雨”或蒸发粒子。
- 乳状云：强天气前兆，适合作为稀有危险视觉。
- 波涛云：大规模波浪扰动。
- 糙面云：混乱、暗沉、危险环境。
- 弧状云：风暴前沿，适合推进型云墙。
- 砧状云：积雨云顶端形态，可作为雷暴核心上层。
- 墙云、尾云、管状云：积雨云专属危险部件。

### 适合做伴生结构的附属云

- 碎片云：破坏云后生成的小云块或小实体。
- 伞状云：覆盖在云塔顶部的护罩结构。
- 缟状云：高空尾迹和风线。
- 流云：从大型云体边缘泄出的云流，可作为流动路径。

## 云生物群系规划

### 积云原野

入门区域，白色、明亮、安全。资源以云碎片、普通云块为主。

玩法：基础采集、云平台搭建、学习气态/固态转化。

### 层云雾海

低层大片雾状云，能见度低，移动变慢。

玩法：迷路、隐藏遗迹、低视距探索、雾中生物。

### 卷云高空

高空稀薄云带，落脚点少，但移动更快。

玩法：高风险高机动路线、冰晶资源、风道旅行。

### 高层云幕

中层灰白云幕，遮光、安静、范围大。

玩法：过渡区域、天气前兆、低光生态。

### 雨层云海

厚重暗云，持续降雨，潮湿和缓慢。

玩法：水资源、湿润材料、持续消耗、雨中敌对生物。

### 高积云群岛

大量中层块状云组成的岛链。

玩法：跳岛、桥接、波状地形、群聚资源。

### 层积云大陆

低层厚云形成相对稳定的大陆。

玩法：建家、村落、云上农业、低危险扩张。

### 积雨云塔

垂直贯穿多层的雷暴云塔。

玩法：雷暴、强对流、Boss、稀有材料、维度阶段目标。

### 晕光卷层云穹

维度高处的薄幕天空层，带日晕/月晕视觉。

玩法：天象、稀有仪式、维度出口或高阶传送机制。

## 云机制方向

### 自然速度收敛

实体进入气态云时，不应通过无限叠加加速度把实体弹飞，也不应靠生硬上限产生突兀手感。

推荐方向：

- 每种气云定义目标水平速度。
- 实体每 tick 自然趋近这个目标速度。
- 竖直速度主要由 drag、风、天气或特殊结构影响。
- 强对流也不直接写成“固定上限”，而是通过扰动、风场、阶段事件表现。

### 气态与固态转化

当前已有 Cloud Walker 和药水固化机制，未来应继续作为核心系统。

扩展方向：

- 不同云种固化后保留自身类型。
- 特定药水或工具可以把云在不同相态之间切换。
- 部分云只在特殊天气或生物群系中稳定存在。

### 云量与天气

现实中云量、降水和光照关联很强，适合做维度环境系统。

扩展方向：

- 云量影响光照和怪物生成。
- 雨层云提高降水概率。
- 积雨云触发雷暴事件。
- 卷层云或火烧云影响天象。

### 云的水循环

云维度可以有自己的水循环资源。

扩展方向：

- 雨层云提供稳定水源。
- 幡状云产生不会落地的虚雨。
- 积雨云提供暴雨和雷暴材料。
- 云凝结器、云瓶、气态转换器形成早期科技线。

### 云压缩空气与释压技术

云维度里的喷气装备不应先被定义成“烧云块换推力”的燃料器具，更适合的方向是：利用云环境完成空气收集、压缩、储存，再通过快速释压获得推进力。

扩展方向：

- 不同云属影响空气压缩效率，而不是直接对应通用燃料值。
- 卷云和高空稀薄云更适合做高机动释压推进。
- 高层云和中层云更适合做稳定充压或环境供能。
- 雨层云更适合做持续型、缓降型气压系统。
- 积雨云可提供高压爆发，但伴随失稳、放电或过载风险。
- 喷气背包可以在云内缓慢充压，在离开云层后释放压缩空气完成短时上升、悬停或位移。
- 后续若引入电力系统，电力更适合作为压缩、稳压、储压和控制系统的辅助驱动，而不是替代这套核心机制。
- 与其他电力 mod 的兼容应建立在 CloudCraft 自己的压缩空气闭环已经完整成立之后。

### 云材料层级

可以按云层和危险度组织材料。

示例：

- 积云碎片：基础材料。
- 冰晶云丝：来自卷云、高云层。
- 潮湿云絮：来自雨层云。
- 雷暴云核：来自积雨云。
- 薄幕云纱：来自卷层云。
- 厚云坯：来自层积云。

## 特殊云与事件方向

### 火成云

来自火山、燃烧、爆炸或高热区域。

玩法：热气、烟尘、火属性材料、危险浮升流。

### 人为云 / 航迹云

由机器、飞行器或玩家设施生成。

玩法：工业化云生产、云轨道、人工天气。

### 森林云

由森林、巨树或植物生态生成。

玩法：自然恢复、云上森林、生命系材料。

### 瀑布云

由水体落差和峡谷地形生成。

玩法：云瀑、垂直通道、隐藏洞口。

### 夜光云

非常高处的稀有云。

玩法：维度顶层、夜间发光、稀有导航点。

### 火烧云 / 霞

日出日落时的色彩云。

玩法：限时天象、特殊合成窗口、稀有视觉事件。

## 后续可继续补充的方向

后续可以在本文继续扩展：

- 云维度地形生成。
- 云生物与生态链。
- 云村落、遗迹、建筑风格。
- 云工具、云装备、云科技线。
- 天气事件和灾害。
- Boss 与维度阶段推进。
- 与主世界天气、水循环、天空高度的联系。
- Fabric / NeoForge 技术实现拆分计划。

## 当前实现建议顺序

1. 先补齐十大云属的基础方块和气态方块。
2. 让每种云都只有少量明确差异：颜色、速度收敛、透明度、掉落、生成环境。
3. 再加入云生物群系，把云种放入不同高度层。
4. 再引入天气和事件：雨层云降雨、积雨云雷暴、卷层云天象。
5. 最后再做维度主线：进入方式、资源循环、Boss、返回方式。

这样可以避免一开始把系统做散，同时保留未来扩展空间。
